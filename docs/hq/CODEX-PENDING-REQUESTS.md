## 진행 방식
1. 코덱스 토큰 도착 시: 위 MED 항목을 **한 번에 하나씩** 해당 앱 저장소에서 Loop(목표→기준→수정→원래계약 재검증→종료)로 실행.
2. 각 항목 완료 시 이 파일에서 `[x]`로 체크하고 결과(PR·검증)를 한 줄 남긴다.
3. HQ 본체 개선(robom 저장소)은 코덱스와 무관하게 Claude가 계속 audit→fix→release.
