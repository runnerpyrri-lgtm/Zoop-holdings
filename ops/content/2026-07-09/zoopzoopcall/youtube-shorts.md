# 줍줍콜 · 유튜브 쇼츠 #1 (30~40초)

## 훅 후보 (첫 3초 — 하나 고르기)
1. "무순위 청약, 남들 다 넣을 때 넌 몰라서 못 넣지."
2. "가점 0점도 되는 '줍줍' 청약, 알림 없으면 무조건 놓침."
3. "이 아파트, 추첨으로 그냥 가져가는 사람들 있어요."

## 대본
(0-3s) [훅] 화면: 마감 임박 카운트다운 확대
"무순위 청약 '줍줍', 가점 필요 없어요. 근데—"
(3-10s) 화면: 여러 공고가 뜨는 목록 스크롤
"전국에서 매일 뜨는데, 접수 딱 하루만 열리는 것도 많아요. 놓치면 끝."
(10-22s) 화면: 앱에서 공고 탭 → 알림 프리셋 토글(1일·3시간·정각)
"줍줍콜은 전국 무순위·잔여세대·취소후재공급을 한곳에 모아줘요.
공고마다 '마감 1일 전·3시간 전·정각' 알림을 탭 한 번으로."
(22-32s) 화면: 알림 울리고 청약홈으로 딥링크 이동
"알림 오면 바로 청약홈으로 연결. 무료예요."
(32-38s) [CTA] 화면: 로고 + URL
"지금 '줍줍콜' 검색. 알림만 켜두세요."
