# v0.1.0 설계 (spec)

## 목표

무순위·잔여세대·취소후재공급 공고를 한 화면에 모으고, 접수 시작·마감 알림이 실제로 폰에서 울리는 것을 사람이 즉시 확인할 수 있는 상태로, 접속 가능한 모바일 URL과 함께 출시한다.

## 완료 기준

1. `pnpm test`/`pnpm typecheck`/`pnpm build` 전부 통과.
2. 공개 URL에서 목록→상세→알림 켜기→내 알림 흐름이 동작.
3. 알림 프리셋(시작 1일/3시간/정각, 마감 1일/3시간/1시간)이 core 순수함수로 계산되고 과거 미예약이 테스트로 잠김.
4. 테스트 알림 버튼으로 실제 브라우저 알림 수신을 5초 안에 확인 가능.
5. 샘플 데이터임이 화면에 명시되고, 실데이터 전환 절차가 문서화됨.

## 아키텍처

- `packages/core` — 플랫폼 무관 순수함수(타입·상태판정·KST·알림계산·API 정규화). Vitest.
- `apps/web` — React+Vite PWA. HashRouter(정적 호스팅 안전), localStorage 상태, 15초 스케줄러, 서비스워커.
- `supabase/functions/notices` — 서비스키 은닉 프록시(10분 캐시). v0.1.0에서는 코드만 준비, 배포는 사람 키 발급 후.

## 화면

목록(필터+접수중/예정/마감 그룹) / 상세(카운트다운+프리셋+청약홈 딥링크) / 내 알림(예약 목록+테스트 알림+권한 안내) / 안내(출처·한계·홈화면 추가).

## 디자인 방향

공고문(관보) 감성: 종이색 배경 + 잉크 네이비 + 인주 레드. 시그니처는 도장형 D-day 스탬프. 본문 Pretendard 17px+(중장년 가독), 제목 고운바탕. 라이트/다크, 모바일 우선 반응형(max 680px 칼럼), 접근성(포커스 링·reduced motion·44px 터치 타깃).
