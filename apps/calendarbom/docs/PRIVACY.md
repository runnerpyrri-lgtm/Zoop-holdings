# 캘린더봄 개인정보 처리 요약

정식 문서: https://robom.kr/privacy/calendarbom

- 캘린더봄은 계정·서버 없이 동작합니다. 일정, 사람 이름·관계(선택), 약 이름, 복용 확인 기록은 사용자의 기기 안(localStorage)에만 저장됩니다.
- 복용 확인은 의학적 건강기록이 아닌 편의용 단순 확인 기록이며, 설정에서 보관 기간(7·30·90일)을 정하거나 모든 데이터를 한 번에 삭제할 수 있습니다(삭제 전 백업 파일 자동 제공).
- 어떤 일정 내용도 외부로 전송하지 않습니다.
- 브라우저 알림 권한은 알람 표시에만 사용합니다.
- 보관 파일(JSON/ICS)은 사용자가 직접 내려받아 관리합니다.
- 폰트(Pretendard)는 jsDelivr CDN 에서 불러옵니다. 이때 브라우저가 CDN 에 IP 주소를 전달할 수 있습니다.
